📖 表電卓の使い方

スプレッドシート風の電卓です。表に数値を入れて、合計・平均・数式や各種モードで素早く計算できます。入力内容は自動保存され、次回も続きから使えます。

✏️ 基本の使い方
セルをタップして選び、テンキーで入力 → Enterで確定（下のセルへ移動）。
上部の入力欄から直接入力／編集もできます。
列見出し（A,B…）や行番号をタップすると、その列・行を選択して下に合計が出ます。
列見出しの右端／行番号の下端をドラッグすると、その列の幅・行の高さを個別に変更できます（ダブルタップで自動に戻す）。
左上のコーナー（⤢）をスワイプ、または表の上で2本指ピンチすると、セル全体の大きさ（表示倍率）をまとめて変更できます（コーナーはダブルタップで100%に戻す）。設定の「セル文字」の＋−でも変更できます。
🔢 テンキーのボタン
CE 入力中の値をクリア　⌫ 1文字消す
± 符号（＋/−）を反転
fx= 数式入力を開始（先頭が = になる）
文字 文字（テキスト）を入力
モード 計算モードを切替
テンキーは数字ページ（標準）を土台に、関数電卓・関数記号・登録ページがその上に重なってスライドインする構成です（数字ページはどこからでもフリック開始可）：
・数字ページを右へフリック＝関数電卓ページが重なって表示…SIN・COS・TAN・ASIN・ACOS・ATAN・LOG・LN・EXP・π・e・n!（階乗）・x²・^（べき乗）・1/(・MOD( などを数式に挿入。DEG/RADボタンで三角関数の角度単位（度数法⇄弧度法）を切替できます。どのボタンの上でも左へフリックすると数字ページに戻ります。
・数字ページを左へフリック＝関数・記号ページが重なって表示…( ) , : ・SUM・AVERAGE・MAX・MIN・SQRT・ABS・ROUND・COUNT・IF・比較記号 などを数式（先頭が =）に挿入。どのボタンの上でも右へフリックすると数字ページに戻ります。
・数字ページを上へフリック＝登録ページが重なって表示…紫のボタンを長押しすると登録画面が開きます。基本は文字・数字を入力して「登録」するだけ（よく使う文字・数字を登録、数値の途中でタップすると連結。例：100→「円」で100円）。「🔧 便利機能から選ぶ（オプション）」を開くと、「Enter」「1文字削除」「戻る」「進む」「保存」「リスト」「リセット」「CSV出力」「CSV読込」「ナイト」「モード」などの便利機能も選んで登録できます。登録後はタップで入力／実行（未登録の「＋」はタップでも登録画面が開きます）。登録画面の「🔀 このボタンを別の場所へ移動」を押すと、移動先のボタンをタップして位置を入れ替えできます。登録は保存されます。どのボタンの上でも下へフリックすると数字ページに戻ります。
テンキー上端（横向きは左端）のバーをドラッグすると大きさ（横向きは幅）を変更できます。設定の「🔧 テンキーのボタン配置を編集」で各ボタンの位置も入れ替えできます。
Σ オート（合計・平均）
Σを押すと隣の連続データを自動で合計し =SUM(範囲) を作成。Enterで確定。
押した後にセルを始点→終点までスライドすると、なぞった範囲を合計範囲にできます（タップで1セル）。
Σを長押しすると「合計／平均」を選べます。選んだ方を記憶し、ボタン表示も変わります。
🧮 数式
先頭を = にすると数式になります。セルは A1 のように指定。
=A1+B1*2 … 四則演算（×÷も可）
=SUM(A1:A5) 合計　=AVERAGE(A1:A5) 平均
他に MAX・MIN・COUNT・ABS・SQRT・ROUND・IF が使えます。
数式入力中にセルをタップするとセル名が挿入されます。
∑ 合計欄（画面下）
列・行を選ぶと合計が表示されます。
「合計」の文字をタップ → 合計⇄平均を切替。
「¥」をタップ → 数字⇄漢数字（兆・億・万）表示を切替。
数値をタップ → その数値をコピー。
🎨 セルの書式
上部の「A1」（セル名）をタップすると、そのセルの小数点桁・太字/斜体/下線・文字色・背景色・大きさを変更できます。
🔒 保護：書式パネルの「保護」をオンにすると、そのセルは編集できなくなります（🔒マークが付きます）。誤って消したくない値の固定に便利。解除はもう一度タップ。
📥 コピー操作
フィル：セルを長押し→スライドで、その値（数式は自動でずらして）を隣へコピー。
コピペ：セルを2回タップでコピー → 別セルを2回タップで貼り付け。
PCでは Ctrl+C / Ctrl+V、元に戻すは Ctrl+Z。
🔀 モード（「モード」ボタン）
切替後も各モードの内容はリセットするまで保持されます。
通常：標準の表　簡易：列少なめ・大きな表示
買物：単価×個数で小計
割り勘：合計を人数で割る
消費税：税抜→税込／税込→税抜＋消費税（税率は設定）
割合計算：元の値の何割・何％と、足す/引くを計算
容積：縦×横×高さ＝容積　予算配分：予算と個数から最適な購入数
時間計算：開始〜終了の時刻から所要時間
単位水量：土研法エアメータ法で生コンの単位水量を推定。表に配合（単位水量W1・セメント・細骨材・粗骨材・混和材・配合空気量）と試験値（容器容積V・容器質量・容器+試料・測定空気量）を入力すると、γ1・γ2から W＝W1＋0.7×(γ1−γ2) で単位水量を推定し、配合値との差を許容差（合格±/限界±、下のバーで変更可）で合否判定します。配合の質量はkg/m³、試験の容器容積・容器質量・容器+試料はg・ccで入力。
⚙️ 設定（このパネル）
行／列：表の大きさ　小数：小数点以下の桁（自＝自動）
▦ デフォルト：タップすると決めておいた行数・列数に一発で変更。長押しすると既定の行数・列数を設定する画面が開きます。
税率：消費税モードの税率
テンキー／セル文字／小計文字：各文字サイズ
🔧 テンキーのボタン配置を編集：押すと編集モードになり、テンキーのボタンをドラッグして好きな位置に入れ替えできます（同じ大きさのボタン同士。Enter・0は固定）。関数ページも編集可能（編集中もEnter上のフリックでページを移動できます）。配置は自動保存され、編集バーの「↺初期化」で両ページを元に戻せます。
🔀 モードの表示/非表示：「モード」一覧に出す標準モードを選べます。チェックを外すと一覧から隠れます（内容は保持。「通常」は常に表示）。
💾 保存・その他（上部メニュー）
保存：名前を付けて保存　リスト：保存データを開く
保存したときのモードも一緒に記録され、読み込むとそのモード（単位水量など）で再現します。リストの各データにはモードのバッジが表示されます。
リストの上に🗂記録／📌タブ1／📌タブ2／🔀モードの4つのタブがあります。保存した記録はまず「記録」タブに入り、各データのタブ選択欄でよく呼び出したい記録をタブ1・タブ2に振り分けて整理できます。タブは長押しで名前を変更できます（例：「現場A」「経理」）。
「モードへ」で記録をモードに登録すると、その記録は自動で「モード」タブへ移動し、誤って削除・改名・タブ移動できないよう保護されます（🔒ロックと同じ状態。手動での「モード」タブへの振り分けはできません）。
リスト画面いちばん下の⬇書き出しで保存データをファイルに出力 → 別の端末・ブラウザで⬆読み込みすると復元できます（バックアップ・引っ越し用）。
各データの🔓ロックをタップすると🔒保護され、誤って削除・改名できなくなります（もう一度タップで解除）。
↩戻る／↪進む：直前の入力を元に戻す／やり直す
📄 CSV出力：表をCSVファイルに書き出し　📥 CSV読込：CSVファイルを今の表に読み込み（上書き）。文字コード（UTF-8／Shift_JIS）は自動判別するのでExcelのCSVもそのまま読めます。どちらも詳細設定内にあります。
🌙：ダークモード切替　リセット：今のモードの内容を消去
